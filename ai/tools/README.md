# ai/tools

Herramientas que los agentes usan para ejecutar tareas (08_AGENT_ARCHITECTURE §"Herramientas"). Ejemplos: Search, Summarize, Graph Search, OCR, Vision, Document Generator, PowerPoint Generator, Excel Generator, Timeline Builder.

**No hace**: no contiene lógica de negocio ni decide cuándo usarse — eso lo decide el agente que la invoca.
