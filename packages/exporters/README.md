Conversión de conocimiento estructurado a formatos de salida (12§6.4).

**Responsabilidad**: generar DOCX (python-docx), PDF (PyMuPDF), PPTX (python-pptx), XLSX (openpyxl), HTML/Markdown.

**No hace**: no decide contenido ni diseño — el contenido viene de `ai/agents`, el diseño de `packages/templates`.
