# packages/templates

Sistema visual corporativo (12§6.3, Template Engine en 02_PRODUCT_ARCHITECTURE).

**Responsabilidad**: logos, colores, tipografía, layouts, componentes de documentos. Toda salida generada por el sistema pasa por aquí.

**No hace**: nunca genera contenido, solo aplica branding a contenido ya producido por `ai/agents` o `packages/exporters`.
